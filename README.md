# Breakthrough Energy (Replica)

React + Tailwind CSS single-page replica inspired by `https://www.breakthroughenergy.org/`.

## Tech

- Vite + React
- Tailwind CSS
- Framer Motion
- Fonts: **Outfit (900)** + **DM Sans**

## Getting started

You need Node.js (LTS recommended). Then:

```bash
cd breakthrough-energy-replica
npm install
npm run dev
```

Build / preview:

```bash
npm run build
npm run preview
```
